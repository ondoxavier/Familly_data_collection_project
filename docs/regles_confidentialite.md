# Règles de confidentialité recommandées

Les données généalogiques peuvent contenir des informations personnelles sensibles. Il faut donc appliquer quelques règles simples dès la collecte.

## Règles principales

1. Informer les participants de l'objectif du projet.
2. Limiter l'accès aux membres autorisés de la famille.
3. Éviter de rendre publiques les informations sur les personnes vivantes.
4. Permettre la correction ou la suppression d'une information.
5. Garder une trace des personnes qui ont fourni les données.
6. Marquer les informations incertaines comme `à vérifier`.
7. Sauvegarder régulièrement la base.

## Données à manipuler avec prudence

- dates de naissance complètes ;
- numéros de téléphone ;
- adresses ;
- situations familiales complexes ;
- enfants mineurs ;
- informations de décès.

## Recommandation MVP

Pour commencer, collecter uniquement ce qui est nécessaire :

- nom ;
- prénom ;
- sexe ;
- lien familial ;
- année ou date approximative de naissance ;
- statut vivant/décédé ;
- remarques optionnelles.
