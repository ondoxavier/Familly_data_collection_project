# Checklist avant d'envoyer les formulaires

- [ ] Identifier tous les chefs de famille.
- [ ] Créer une branche pour chaque chef de famille.
- [ ] Générer un lien unique par branche.
- [ ] Envoyer le message WhatsApp avec le lien.
- [ ] Expliquer que les dates approximatives sont acceptées.
- [ ] Prévoir une personne référente pour aider ceux qui ne sont pas à l'aise avec le numérique.
- [ ] Vérifier les doublons après réception.
- [ ] Marquer les informations incertaines comme `to_verify`.
- [ ] Exporter les données validées.
- [ ] Préparer l'import Neo4j.
